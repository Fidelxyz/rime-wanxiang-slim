# === 文档 ===
# https://fidel.js.org/rime-wanxiang-slim/customization/dictionaries#%E7%94%A8%E6%88%B7%E7%9F%AD%E8%AF%AD 
#
# === 支持转义字符 ===
# \n - 换行
# \s - 空格
# \t - 制表符
# \\ - 反斜杠
#
# === 格式 ===
# 一行代表一个用户短语，格式为：`词汇<Tab>编码<Tab>权重`，以制表符分隔。
# 权重为可选项。
#
# === 示例 ===
# 吧	b	5
# 静夜思\n\t李白\n床前明月光\n疑似地上霜\n举头望明月\n低头思故乡	jys	5
#
# === 此行后不得有注释 ===
