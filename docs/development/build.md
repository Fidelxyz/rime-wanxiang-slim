# 构建

`scripts/build.sh` 是用于构建的自动化脚本。

该脚本会生成多个方案包，包含基础版（Base）和对应不同辅助码的进阶版（Pro）。构建产物位于 `dist` 目录下。

## 前置要求

- Bash
- Python 3
- rsync
- zip（可选，使用 `--no-archive` 参数时不需要）

## 基本用法

在项目根目录下运行：

```bash
bash scripts/build.sh
```

该脚本会：

1. 生成进阶版词库
2. 打包所有方案（基础版 + 对应不同辅助码的进阶版）
3. 将打包结果输出到 `dist` 目录
4. 为每个方案创建 `.zip` 压缩包

## 命令行参数

### `--no-archive`

跳过创建压缩包，仅生成未压缩的方案目录。

> [!TIP]
> 开发调试时可利用该参数加快构建速度。

### `--no-dicts`

跳过词库文件的构建，构建产物中将不包含词库文件。

> [!TIP]
> 开发调试时可利用该参数加快构建速度。
