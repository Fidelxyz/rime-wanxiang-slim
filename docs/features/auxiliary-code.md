# 辅助码

在输入状态下，输入以下命令切换对应直接辅助或间接辅助。切换后需**重新部署**。

| 命令 | 辅助码引导模式 |
| --- | --- |
| `/jjf` | 间接辅助 |
| `/zjf` | 直接辅助 |

## 直接辅助码 <Badge type="tip" text="仅 Pro" />

在双拼编码后直接追加辅助码。

重码处理：

- 当双拼 + 辅码与现有词组重码时（均为 4 码），优先显示词组。
- 在编码末尾追加 `/` 可强制聚拢辅助码，优先展示单字。
- 以大写输入任意一位辅助码，即可强制聚拢为单字。

::: info 示例

以「你」字为例：双拼编码为 `ni`，辅助码为 `re`。

可使用以下方式输入：

- 双拼 `ni` → `ni`。
- 双拼 `ni` + 首位辅助码 `r` → `nir`。
- 双拼 `ni` + 完整辅助码 `r` → `nire`。
- 双拼 `ni` + 完整辅助码 `re` + 强制聚拢 `/` → `nire/`。
- 双拼 `ni` + 完整辅助码首位大写 `Re` → `niRe`。
- 双拼 `ni` + 完整辅助码末位大写 `rE` → `nirE`。
- 双拼 `ni` + 完整辅助码大写 `RE` → `niRE`。

:::

## 间接辅助码 <Badge type="tip" text="仅 Pro" />

在双拼编码后输入引导符 `/` ，并追加辅助码。

::: info 示例

以「你」字为例：双拼编码为 `ni`，辅助码为 `re`。

可使用以下方式输入：

- 双拼 `ni` → `ni`。
- 双拼 `ni` + 引导符 `/` + 首位辅助码 `r` → `ni/r`。
- 双拼 `ni` + 引导符 `/` + 完整辅助码 `re` → `ni/re`。

:::
